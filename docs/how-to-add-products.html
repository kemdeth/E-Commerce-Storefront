How to Add a
New Product

All products are stored in a single PRODUCTS array inside index.html. No build step, no external files — just edit and refresh.

Open index.html and find this comment inside the <script> tag:

JS 
// DATA
const PRODUCTS = [ ... ]

Scroll to the end of the array and paste your new product object before the closing ].

1
Copy the product template

Paste this at the end of the PRODUCTS array and fill in your values.

JS 
{
  id: 7,                            // must be unique — increment from last id
  name: "Your Product Name",
  price: 20,                          // number only, no $ sign
  description: "Short description of what this product includes.",
  category: "UI Kits",              // see Step 2 for valid values
  icon: "🎨",                        // any emoji works
  iconBg: "rgba(52,211,153,0.12)",  // icon box background — see Step 4
  iconColor: "#34d399",            // icon / emoji tint — see Step 4
  badge: "new",                     // "popular" | "new" | "sale" | null
  tags: ["Figma", "Dark Mode"],     // 2–3 short tags
},
2
Choose the right category

The category value controls which filter pill shows the product. It must match exactly — capital letters and spacing included.

Value to use Shows under filter
"UI Kits" UI Kits pill
"Templates" Templates pill
"Icons" Icons pill
💡 All three categories also appear under the All filter — you never need to add a new category value.
3
Pick a badge (or skip it)

The badge field shows a small label in the top-right corner of the card. Use null if you don't want one.

Value Preview Color
"popular" Popular Purple
"new" New Green
"sale" Sale Red
null No badge —

✓
Before you save

Run through this quickly before refreshing the browser.

  • id is unique — not used by any other product in the array.
  • category matches one of the 3 valid values exactly (capital letters matter).
  • There is a comma , after the closing } of your object (unless it is the last item).
  • badge is either "popular", "new", "sale", or null.
  • price is a plain number — no $ sign, no quotes.
⚡ No build step needed. Save index.html and refresh the browser — your new product will appear immediately.
